basic

variable

  • let name; to set variable, default value if undefined.
  • can't use reserved keyword
  • need to be meaningful
  • cannot start with a number
  • cannot contain space of hyphen(-)
  • are case-sensitive

constants

  • const interestRate = 0.3; constant cannot be re-assign

primitive type

  • string: let name = 'Vicki'
  • number: let age = 25;
  • boolean: isApproved = false;
  • undefined: let firstName = undefined; normally if we don't assign value, it is undefined
  • null: used when we want to clear the value of a variable

dynamic typing

  • dynamic typing mean the type of variable can change at run time
  • only one type for number no float etc
  • undefined is a type but also a value

reference type

  • object

object

  • think of object as a person, which can have multiple feature
  • to access the object feature, dot notation or bracket notation

arrays

  • a data structure, used to store a list of item

operator

javaScript operator

  • use operator to create expression to create algorithm

arithmetic operator

  • x - y
  • x + y
  • x * y
  • x / y
  • x % y
  • x ** y
  • console.log(x++): show 10 but become 11
  • console.log(++x): show 11
  • console.log(x--): show 10 but become 9
  • console.log(--x): show 9

assignment operator

  • x += 5 -> x = x + 5
  • x *= 5 -> x = x * 5

comparison operator

  • relational: >, <, >=, <=,
  • equality: ===, !==

equality operator

  • strict equality: same type and same value
  • lose equality: same value different type
  • console.log(true == 1); return true

ternary operator

  • start with condition if condition is true use first value else use second value
  • if a customer has more than 100 Point, they are gold else they are sliver
  • let type = point > 100 ? "gold" : "silver";

logical operator

  • logical AND (&&): return true if both operands are TRUE
  • logical OR (||): return true if one of the operand is true
  • NOT (!): return the opposite

logical operator with non-boolean

  • result of logical operator not necessary to be true or false depend on the operands
  • falsy: undefined, null, 0, false, '', NaN(not a number)
  • truthy: anything not falsy
  • short circuiting: false || 2 || 1: return 2, get to truthy return truthy and ignore the rest

bitwise operator

  • 1 = 00000001, 2 = 00000011
  • 1 | 2 = 3: as it return 00000111
  • 1 & 2 = 0: as it return 00000000
  • example use for access control system

control flow

if...else

  • if (condition) { statement } else if (condition) { statement } else { statement }
  • if hour is between 6am and 12pm: good morning;
  • hour is between 12pm and 6pm: good afternoon;
  • otherwise: good evening;

switch...case

  • switch (role) { case 'guest': console.log('Guest User') break; //if not break it will execute to next block case 'moderator': console.log('Moderator User') break; }
  • if hour is between 6am and 12pm: good morning;
  • hour is between 12pm and 6pm: good afternoon;
  • otherwise: good evening;

for loop

  • for (let i = 0; i < 5; i++) { console.log(i); }
  • 0 1 2 3 4
  • use % to get odd/even, start from the back with-- to reverse order
  • i only accessible inside the loop
  • return in a loop will exit the loop

value vs reference type

  • Primitives are copied by their value
  • objects are copied by their reference, when copy an object, we copied it's address so when we change the value using that address, the changes will reflect to other variable as well
  • let x = 10; let y = x; x = 20; ==> only change the value of x, value of y will still be 10
  • let x = {value: 10}; let y = x; x.value = 20;
  • value type: number, string, boolean, symbol, undefined, null
  • reference type: object, function, array
